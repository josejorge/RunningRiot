## Description
RunningRiot is an experimental mobile experience that bridges web-based gaming with native Android functionality. This project serves as a dedicated Android Wrapper (WebView) for the HTML5 game RunningRiot, optimized for performance and mobile accessibility.

## Key Features
Full-screen WebView integration: Seamlessly renders HTML5 canvas.

Splash Screen: Custom branded loading experience.

Optimized Performance: JavaScript enabled with hardware acceleration support.

Navigation Handling: Integrated Android back-button support for in-game navigation.

## Tech Stack
Android Studio (Java/Kotlin)

Android WebView API

HTML5 / JavaScript (Game Core)
